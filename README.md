## ðŸš€ Getting Started

### Prerequisites

- Node.js (>= 14.x)
- MongoDB

### Installation

#### Clone the Repository

```sh
 git clone https://github.com/Bhupesh-Khordia/E-Commerce.git
 cd E-Commerce
```

#### Frontend Setup

```sh
 cd frontend
 npm install
 npm run dev
```

#### Backend Setup

```sh
 cd backend
 npm install
 npm start
```

#### Admin Setup

```sh
 cd admin
 npm install
 npm run dev
```
